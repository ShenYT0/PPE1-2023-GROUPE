      食品 衛生 監視員 の 仕事 内容 は ？ 



食品 衛生 監視員 と は 日本 国民 の 食 の 安全 を 守る エキスパート 



食品 衛生 監視員 に は 国家 公務員 と 地方 公務員 が いる 



・ 食品 衛生 監視員 の 仕事 は ２ つ の 領域 に 分かれる 



      「 食品 衛生 監視員 」 は 「 食品 衛生 法 」 など の 法律 に 基づい て 、 私たち が 口 に する 食品 の 安全 と 安心 を 守る スペシャリスト です 。 

      空港 や 港 など の 検疫所 で 働く 「 国家 公務員 」 の 食品 衛生 監視員 と 、 自治体 の 保健所 で 働く 「 地方 公務員 」 の 食品 衛生 監視員 が い ます 。 

      それぞれ の 仕事 内容 を 紹介 し ます 。 　 



・ 空港 や 港 など の 検疫所 で の 仕事 と は ？ 



      日本 で は 毎日 大量 の 食品 が 海外 から 輸入 さ れ て い ます 。 

      もはや 輸入 食品 なし に 国民 の 生活 が 成り立た ない ほど です 。 

      そう し た 輸入 食品 に よる 食 中毒 や 感染症 など の 危険 を 防ぐ ため 、 輸入 食品 の 安全 確保 は 非常 に 重要 な 仕事 です 。 

      その ため 日本 で は 輸入業 者 は 食品 を 輸入 する 時 に は 「 輸入 届出 」 を する こと が 義務 づけ られ て い ます 。 

      届出 を 受け た 厚生 労働省 の 検疫所 で は 、 食品 衛生 監視員 が 、 その 食品 の 審査 を 行っ て 安全性 を 監視 し て い ます 。 

      空港 や 港 の 検疫所 で 働く 「 国家 公務員 」 の 食品 衛生 監視員 の 仕事 は 主に 次 の ３ つ に 分け られ ます 。 

      （ １ ） 輸入 食品 の 監視 を 行う 

      輸入業 者 は 、 全国 の 検疫所 に 配置 さ れ た 「 輸入 食品 届出 窓口 」 に 輸入 届け を 提出 し なけれ ば なり ませ ん 。 

      それ を 食品 衛生 監視員 は 届出 に 書か れ た 内容 を 見 て 、 「 食品 衛生 法 に 合っ て いる か 」 「 きちんと 検査 が 行わ れ て いる か 」 など を チェック し ます 。 

      日本 で 禁止 の 添加物 が 使用 さ れ て いる 場合 など が あれ ば 業者 に 指導 し ます 。 

      また 多種 多様 な 輸入 食品 を 幅広く 監視 する ため に 、 湾口 （ わん こう ： 湾 の 外海 へ の 出入り 口 ） 地域 や 倉庫 に 自ら 出かけ て 、 倉庫 で 輸入 食品 の 確認 を し たり 、 検 

      査 の ため に 一部 を 抜き取っ たり し て い ます 。 

      検査 の ため に 抜き取ら れ た 食品 は 精密 機器 を 使っ て 検査 分析 が 行わ れ ます 。 

      これ を 「 モニタリング 検査 」 と いい ます 。 

      （ ２ ） 検査 （ 試験 分析 ） を 行う 

      食品 衛生 監視員 は 、 書類 の チェック だけ で なく 精密 機器 を 用い て 検査 分析 も 行い ます 。 

      検査 に は 、 微生物 検査 と 理化学 検査 が あり ます 。 

      微生物 検査 で は 輸入 食品 が 微生物 に よっ て 汚染 さ れ て い ない か 、 理化学 検査 で は 、 残留 農薬 や 残留 動物 用 医薬品 （ 抗菌性 物質 等 ） 、 添加物 など の 有毒 有害 物質 

      など が 含ま れ て い ない か 検査 し ます 。 

      （ ３ ） 人間 の 感染症 の 病原体 の 侵入 を 防ぐ 

      食品 衛生 監視員 が 検査 する の は 輸入 食品 だけ で は あり ませ ん 。 

      国内 に 常在 し ない 感染症 ・ 検疫 感染症 の 病原体 の 侵入 を 防ぐ ため 、 海外 から 来る 船舶 や 飛行機 、 乗客 や 乗組員 など も 検査 し ます 。 

      感染 患者 を 発見 し たら 隔離 や 消毒 など の 措置 を 行う 場合 も あり ます 。 また 輸入 する 動物 の 審査 も 行っ て い ます 。 



・ 自治体 の 保健所 に おけ る 仕事 と は ？ 



      一方 、 自治体 の 保健所 で 働く 「 地方 公務員 」 の 食品 衛生 監視員 の 仕事 は 以下 の 通り です 。 

      1   飲食店 など の 衛生 チェック と 許可 事務 を 行う 

      飲食店 や 食品 の 製造 販売店 を 始める に は 、 「 食品 衛生 法 」 や 自治体 の 営業 許可 や 届出 が 必要 です 。 

      食品 衛生 監視員 は 、 飲食店 や 販売店 の 施設 が 、 食品 衛生 法 を 守っ て 安全 で ある か を 厳しく チェック し 、 営業 許可 事務 を 行い ます 。 

      2   飲食店 の 立ち入り 調査 

      食品 衛生 監視員 に は 、 飲食店 など に 対し て 、 監視 や 指導 の ほか 、 立ち入り 調査 を 行う 権利 が あり ます 。 

      その 際 、 検査 の ため に 、 その お店 で 提供 し て いる 必要 最小 量 の 食品 や 添加物 を 抜き取っ て 持ち帰る こと が あり ます 。 

      3   食品 の 検査 と 販売 停止 など の 処分 の 決定 

      抜き取り 調査 で 飲食店 など の 営業 施設 から 持ち帰っ た 食品 を 検査 し て 、 食品 衛生 法 の ルール が 守ら れ て い ない 場合 は 、 販売 停止 ・ 営業 停止 など の 行政 処分 を 決 

      定 する こと も あり ます 。 

      4   食 中毒 など の 調査 や 対策 

      食 中毒 など が 発生 し た 場合 は 、 関係者 や 医師 の 協力 の もと で 該当 食品 の 検査 を 行い ます 。 

      そして 販売 の 中止 を 決め たり 、 営業 停止 の 行政 処分 を 決定 し たり し ます 。 

      再発 防止 の ため の 指導 や 対策 も 行い ます 。 

      5   食品 衛生 に 関する 講習会 を 開く 

      その ほか 、 事業者 を 対象 に し た 食品 衛生 に 関する 講習会 を 開き 、 地域 の 人 たち に 安全 な 食品 が 提供 さ れる よう に 取り組ん で い ます 。 

      「 ○ ◯ 店 で 買っ た 食品 に カビ の よう な もの が 生え て いる 」 など の クレーム が き た 場合 は 調査 を し 、 問題 が 発見 さ れ た 場合 は 再発 防止 の 対策 を 行い ます 。 

      食品 衛生 監視員 は どんな 働き方 を する の ？ 



食品 衛生 監視員 は 基本的 に は 週休 2 日 制 で 働く 



      前述 の よう に 、 食品 衛生 監視員 は 「 国家 公務員 」 か 「 地方 公務員 」 に 分け られ ます 。 

      主に 国家 公務員 は 空港 や 港 など の 検疫所 で 働き 、 地方 公務員 は 主に 自治体 の 保健所 で 働き ます 。 

      ほか に も 厚生 労働省 や 都道府県 庁 、 卸売 市場 の 検査 所 など 、 活躍 の 場 が あり ます 。 

      食品 衛生 監視員 は 公務員 な の で 、 基本的 に は 週休 2 日 制 で 、 原則 土 ・ 日 祝日 や 年末 年始 は 休み と なっ て い ます 。 

      しかし どの 勤務地 に 配属 さ れる か に よっ て 勤務 体系 など は 大きく 異なり 、 勤務 時間 など は 一概 に は 言え ない よう です 。 

      また 空港 の 検疫所 など は 年中 無休 で 稼働 し て いる ため 、 交代制 勤務 の 職場 で は 、 夜 に 出勤 し たり 、 土日 や 祝日 で も 勤務 に なっ たり する 場合 も あり ます 。 

      また 国家 公務員 の 食品 衛生 監視員 は 、 全国 に ある 検疫所 に 、 2 年 から 3 年 ごと に 異動 が あり ます 。 

      食品 衛生 監視員 は どんな 人 に 向い て いる の ？ 



食品 衛生 監視員 は 「 食 の 安全 」 に 興味 が 強い 人 に 向く 



      コミュニケーション 能力 の 高い 人 

      検疫所 で 輸入業 者 と 接し たり 、 地域 の 飲食店 の 店主 と 接し て 指導 し たり する など 、 食品 衛生 監視員 は コミュニケーション 力 が 欠かせ ませ ん 。 

      また 食品 の 調査 や 分析 を チーム で 行う の で 協調性 も 大切 です 。 

      「 食 の 安全 」 に 対する 興味 が 強い 人 

      食品 に 関する 法律 や 食品 の 試験 検査 技術 など 、 さまざま な 専門 知識 や 技術 が 必要 と さ れる ため 、 「 食 の 安全 」 に 対する 強い 興味 関心 を 持っ て いる こと が 必須 で 

      す 。 

      また 近年 で は 、 海外 から 輸入 さ れる 食品 の 種類 も 増え て おり 、 検査 に つい て の 法令 も 日々 変化 し て いく の で 常 に 最新 の 情報 に アンテナ を 張る こと が 大事 です 。 

      社会 貢献 の 気持ち や 責任感 が 強い 人 

      食品 衛生 監視員 は 国民 や 地域 住民 の ため に 貢献 する と いう 奉仕 の 精神 が 必要 です 。 

      「 日本 の 食 の 安全 は 自分 が 守る 」 と いっ た 強い 責任感 が ある 人 に 向い て い ます 。 

      食品 の 検査 や 分析 が 好き な 人 

      輸入業 者 の 倉庫 や 、 飲食店 の 店舗 から サンプリング し た 食品 の 検査 や 試験 分析 を 行う こと も 重要 な 業務 です 。 

      微生物 の 検査 や 、 有毒 物質 の 理化学 検査 に 興味 が ある 人 、 実験 の 大好き な 人 に 向い て いる 仕事 と 言え ます 。 

      食品 衛生 監視員 の 将来 展望 は ？ 



食品 衛生 監視員 の 将来 の 展望 は ？ 



      食品 衛生 監視員 は 、 国民 の 健康 の ため に 食 の 安全 と 安心 を 守る 仕事 です 。 

      残留 農薬 、 添加物 、 放射性 物質 、 残留 動物 用 医薬品 （ 抗菌性 物質 等 ） など の 悪影響 や 、 疫病 感染 の 危険 から 国民 を 守る ため に 、 今 も 将来的 に も 必要 と さ れる 仕 

      事 と 言え ます 。 

      食品 衛生 監視員 に は こう すれ ば なれる ！ 



食品 衛生 監視員 に なる に は ？ 試験 を 受ける ため の 「 受験 資格 」 が 必要 



      食品 衛生 監視員 に なる に は 、 食品 衛生 監視員 の 公務員 試験 を 受ける ため の 「 受験 資格 」 を 得る 必要 が あり ます 。 

      その 受験 資格 は 、 次 の （ 1 ） から （ 4 ） の どれ か １ つ で も 該当 すれ ば 得 られ ます 。 

      （ １ ） 都道府県 知事 の 登録 を 受け た 食品 衛生 監視員 の 養成 施設 で 、 所定 の 課程 を 修了 し た 人 

      （ ２ ） 大学 や 専門 学校 で 医学 、 歯学 、 薬学 、 獣医学 、 畜産学 、 水産学 また は 農芸 化学 の 課程 を 修め て 卒業 し た 人 

      （ ３ ） 医師 、 歯科 医師 、 薬剤師 また は 獣 医師 

      （ ４ ） 栄養士 で 2 年 以上 食品 衛生 行政 に 関する 事務 に 従事 し た 経験 を 有する もの 

      これ から 高校 を 卒業後 、 進学 し て 食品 衛生 監視員 を めざそう と 考え て いる 人 に とっ て は （ 1 ） また は （ ２ ） を 満たす 進路 を 選択 し て 、 「 食品 衛生 監視員 採用 試 

      験 」 の 受験 資格 を 得る こと が 必要 です 。 

      ※ （ １ ） の 養成 施設 と し て 指定 さ れ て いる 大学 など は 、 厚生 労働省 の ホームページ で 最新 の 情報 を 確認 する こと が でき ます 。 

      上 の 方法 で 受験 資格 を 得 たら 、 次 に 国家 公務員 、 また は 地方 公務員 と し て 食品 衛生 監視員 に 採用 さ れる 必要 が あり ます 。 それぞれ の 採用 方法 を 紹介 し ます 。 



・ 国家 公務員 の 食品 衛生 監視員 に なる に は ？ 



      国家 公務員 の 食品 衛生 監視員 に なる に は 、 人事院 が 行う 専門職 試験 で ある 「 食品 衛生 監視員 採用 試験 」 を 受験 する 必要 が あり ます 。 

      試験 の 合格者 は 1 年間 有効 の 「 採用 候補者 名簿 」 に 得点 順 に 記載 さ れ 、 この 名簿 の 中 から 採用者 が 決定 さ れ て 、 全国 の 主要 な 港 ・ 空港 の 検疫所 など へ 配置 さ れ 

      ます 。 



・ 地方 公務員 の 食品 衛生 監視員 に なる に は ？ 



      地方 公務員 の 食品 衛生 監視員 に なる に は 、 各 都道府県 の 職員 採用 試験 に 合格 する こと が 必要 です 。 採用 試験 の 内容 や 採用 条件 は 各 都道府県 に よっ て 異なり ます 

      。 

      採用 後 、 各 自治体 の 保健所 など の 食品 衛生 に 関する 部局 に 配属 さ れ て 、 都道府県 知事 など から 任命 さ れる と 「 食品 衛生 監視員 」 と し て 働く こと が でき ます 。
