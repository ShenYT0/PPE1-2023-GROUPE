      安全 第 一 ： オーストラリア の ケシ の 種 に よる 中毒 の リコール 、 2022 年 の APAC 食品 安全 記事 トップ 10 、 キムチ の 抜き打ち 検査 



      Related   tags   Japan 



      今回 の 「 Safety 

      First 」 で は 、 オーストラリア の ケシ の 実 の 中毒 に よる リコール 、 2022 年 の APAC 食品 安全 に 関する トップ 10 記事 、 キムチ の 抜き打ち 検査 など を 

      特集 し て い ます 。 



ポピー シード パニック ： オーストラリア の 食品 会社 は 、 全国的 な リコール の 中 で 、 中毒 事件 が 「 ブランド の コントロール の 及ば ない ところ 」 で ある と 強調 ​​ 



      ケシ の 実 の 回収 の 影響 を 受け た オーストラリア の 食品 会社 は 、 政府 が 食品 グレード で は ない ケシ の 実 が 「 誤っ て 」 サプライ チェーン に 混入 し た と 警告 する 中 、 

      事態 は 自分 たち の 手 に 負え ない もの で あっ た と 述べ て いる 。 



      オーストラリア で は 、 ケシ の 実 の 摂取 に 関連 し た 一連 の 重篤 な 食 中毒 事件 が 発生 し て おり 、 30 人 以上 の 消費者 が 、 非 食品 グレード の ケシ の 実 に 含ま れる 有毒 

      化学 物質 で ある テバイン の 高 濃度 摂取 に より 入院 また は 治療 を 受け て いる 。 



      症状 と し て は 、 呼吸 困難 、 筋肉 の 痙攣 、 頻脈 、 発作 、 心停止 、 意識 不明 など が 確認 さ れ て いる 。 ​ 



安全 かつ 確実 ：​ 2022 ​ 年 の ​ APAC ​ 食品 ・ 飲料 業 界 ​ の 食品 安全 に 関する トップ ​ 10 ​ 記事 ​​ 



      シンガポール の 2030 年 食糧 安全 保障 目標 の 課題 、 タイ の 大麻 の 安全性 の 懸念 、 韓国 の チョウマメ 不安 など の ニュース を 取り上げ 、 2022 年 に APAC の 

      食品 ・ 飲料 業 界 で 最も 読ま れ た 食糧 安全 ・ 安心 に 関する トップ 10 記事 を 紹介 し ます 。 



キムチ シーズン 到来 ： 韓国 、 不純物 混入 の 懸念 で メーカー チェック を 実施 ​​ 



      韓国 当局 は 、 衛生 や 不純物 の 問題 に 対する 国民 の 懸念 の 中 、 生産 の ピーク シーズン を 前 に 、 約 2,000 の キムチ および 関連 食材 メーカー に 対し て 食品 安全 チ 

      ェ ック を 実施 し た 。 



      キムチ は 韓国 で は 主食 と さ れ 、 消費者 は 毎食 の よう に おかず と し て 食べ て いる 。 この 製品 の 主な 仕込み と 生産 シーズン は 、 例年 11 月 と なっ て いる 。 



茹で 落花生 が 子ども の ピーナッツ アレルギー 克服 に 役立つ 可能性 が 臨床 試験 で 示さ れる ​​ 



      オーストラリア の フリンダース 大学 と SAHMRI で 行わ れ た 臨床 試験 に よる と 、 ピーナッツ を 最大 12 時間 茹でる と 、 子供 の アレルギー 反応 を 克服 できる 可 

      能 性 が ある 。 



      被験者 に は 茹で た ピーナッツ 、 続い て ロースト し た ピーナッツ が 与え られ た 。 その 結果 、 ピーナッツ アレルギー の 子供 の 80 % も が ピーナッツ を 食べる こと に 

      鈍感 に なる こと が わかっ た 。 



      この 試験 は 、 熱 が ピーナッツ の タンパク質 構造 と アレルギー 特性 に 影響 を 与え 、 重度 の アレルギー 反応 を 引き起こし にくく する こと を 示し た 過去 の 研究 に 基 づ 

      い て いる 。 



有効 期限 実験 ： 韓国 、 食品 ・ 飲料 の 消費 期限 を 設定 する 全国的 な プロジェクト に 着手 ​​ 



      韓国 政府 は 、 国民 の 食品 廃棄物 を 削減 する ため 、 約 2,000 品目 の 食品 ・ 飲料 に 「 消費 期限 」 を 設定 する 全国 プロジェクト を 実施 し た 。 



      2021 年 8 月 、 韓国 は 食品 表示 ・ 広告 規制 の 歴史的 変更 を 可決 し 、 既存 の 「 流通 期限 」 を 新しい 「 消費 期限 」 制度 に 置き換える と いう 、 約 40 年 ぶり の 大き 

      な 変化 を 遂げ た 。 



      新しい 消費 期限 は 、 食品 が 消費 に 安全 で ある と 考え られる 時間 枠 の 60 % から 70 % 以内 から 80 % から 90 % 以内 に 、 製品 が 棚 に 残る こと が 許さ れる 時間 枠 

      を 大幅 に 延長 する 。
