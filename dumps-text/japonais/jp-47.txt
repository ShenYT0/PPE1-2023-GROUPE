食 の 安全 に 関する 検査 なら お 任せ ください 。 



      ＬＳＩ メディエンス は 日本 の 3 大 臨床 検査 センター の ひと つ と 言わ れ て い ます 。 私たち 「 食 の 安全 サポート 事業 」 は 、 フード チェーン 全般 に わたっ て 、 微生物 

      検査 、 理化学 及び 残留 農薬 検査 など の 「 食材 ・ 食品 に 対する 検査 」 や 「 従業員 の 衛生 管理 、 食品 施設 に 対する 検査 」 など 、 食 の 安全 に 関する 多彩 な サービス を 

      提供 し て い ます 。

              検査 項目 を みる

              から 選ぶ 



          *   微生物 検査 

              食材 ・ 食品 に 対する 検査   微生物 検査 

              多彩 な 検査 項目 で 、 食品 内 の 食 中毒 菌 の 有無 を 検査 いたし ます 。 

          *   理化学 検査 

              食材 ・ 食品 に 対する 検査   理化学 検査 

              表示 に 必要 な 栄養 成分 検査 を はじめ 、 アミノ 酸 、 脂肪酸 、 糖類 、 食品 添加物 、 ペット フード 検査 等 、 多様 な 検査 メニュー を ご 提供 いたし ます 

          *   残留 農薬 検査 

              食材 ・ 食品 に 対する 検査   残留 農薬 検査 

              作物 別 セット や 輸入 食品 用 セット など 、 目的 別 の メニュー を 多数 ご 用意 。 迅速 な 検査 を 実施 し ます 。 

          *   ノロウイルス 検査 （ 食材 ） 

              食材 ・ 食品 に 対する 検査   ノロウイルス 検査 （ 食材 ） 

              ウイルス性 食 中毒 の 主な 原因 ウイルス と さ れる ノロウイルス の 検出 を 行い ます 。 

          *   遺伝子 組換え 作物 の 分析 

              食材 ・ 食品 に 対する 検査   遺伝子 組換え 作物 の 分析 

              GMO が 含ま れ て いる か どう か 、 定性 分析 また は 定量 分析 を 実施 いたし ます 。 

          *   コメ の 品種 識別 

              食材 ・ 食品 に 対する 検査   コメ の 品種 識別 

              お 米 の DNA を 抽出 し 、 品種 の 識別 を 行い ます 。 

          *   環境 衛生 調査 

              人 ・ 施設 に 対する 検査   環境 衛生 調査 

              食品 の 調理 ・ 加工 施設 の 環境 衛生 を 定期的 に 調査 致し ます 。 

          *   HACCP ・ GMP システム 構築 支援 

              人 ・ 施設 に 対する 検査   HACCP ・ GMP システム 構築 支援 

              総合的 な 食品 の 衛生 管理 システム の 構築 を お 手伝い いたし ます 。 

          *   従業員 の 衛生 管理 

              人 ・ 施設 に 対する 検査   従業員 の 衛生 管理 

              食品 取扱者 および 調理 ・ 加工 従事者 等 に おい て は 、 食 中毒 菌 に よる 二 次 感染 ／ 汚染 を 未然 に 防止 する ため の 検便 検査 が 不可欠 です 。
