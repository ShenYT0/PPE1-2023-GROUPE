団体 設立   母親 の 輪 広がる 



      　 学校 給食 に 自然 栽培 や 有機 栽培 など の オーガニック 農産物 を 使う 自治体 が 増え て いる 。 北陸 で は まだ 少ない 中 、 富山県 滑川 （ なめ り か わ ） 市 が 市内 全 小 中 ９ 

      校 で オーガニック 米 ご飯 給食 を 始め た 。 立役者 は 一人 の 主婦 。 子ども たち に 安心 し て 食べ て もらえる 給食 を 求め て 市民 団体 も 立ち上げ た 古田 貴子 さん （ ４１ ） 

      。 情熱 と 行動力 が 市 を 動かし 、 母親 たち の 輪 も 広げる 。 （ 中島 健二 ） 

      　 滑川市 に は 「 学校 給食 なめ り かわ の 日 」 が 毎年 １１ 月 に ある 。 地元産 食材 を 積極 活用 する 日 だ が ２０２１ 年 の その 日 、 ご飯 給食 に 初めて 有機米 が 使わ れ た 。 

      古田 さん の 働き掛け だっ た 。 

      　 「 娘 の アレルギー が きっかけ でし た 。 小学校 に 上がる 時 、 弁当 に しよう と 思っ た ん です が 娘 が みんな と 同じ 給食 に し たい と 言う 。 学校 に 対応 を 求め たら 病院 

      で の 診察 が 必要 だ と いう し 、 自分 で 自然 栽培 の 野菜 づくり を し て い て その 良 さ も 知っ て い た 。 給食 を 変えよう と 考え た 。 簡単 に 変わる と 思っ た ん です が … 」 

      　 市長 と 語る 会 に 出 て も 発言 の 機会 が もらえ ず 、 市 の 関係 部署 を 回っ て も 門前 払い 。 交流 サイト （ ＳＮＳ ） で 食 の 安心 安全 問題 を 発信 する しか なかっ た 。 

      　 「 その うち 、 先進地 で 有機 給食 を 実現 さ せ た と いう 人 と 知り合い 、 相談 する と 『 議員 に 言え ば いい 』 と 教え られ た 。 ３ 日 後 、 量販店 で 見かけ た 市議 に 思い 切 

      って 訴え た 。 農薬 や 除草剤 に は 問題点 が ある から 給食 を 変え たい 、 と 。 する と 『 近く 議会 ある から 言う ちゃ 』 と 言わ れ まし た 」 

      　 市議会 で 取り上げ られる と 間 も なく ２１ 年度 の 給食 用 有機米 買い取り の 予算 化 が 決定 。 思っ て い た こと が 分かっ て もらえ た 。 

      　 そんな 農業 と 食 の 問題点 を 描き出し た 映画 「 食 の 安全 を 守る 人々 」 の 上映会 が 昨年来 、 県 内 で 続々 開か れ て いる 。 農薬 や 除草剤 を 題材 に 農産物 の 安全性 を 問 

      う 内容 は 母親 たち の 間 で 大きな うねり を 起こす まで に 。 これ も 古田 さん の 仕掛け だっ た 。 

      　 「 東京 で その 映画 を 見 て 富山 の 人 に も 見 て ほしい と 思っ た ん です 。 滑川 で 開い たら 大 反響 で 県内 全域 に 広がっ て いき まし た 。 それ だけ 食 の 安全 に 関心 の ある 

      人 が いる と 気付き まし た 」 

      　 ただ 、 関心 が あっ て も どう すれ ば 給食 を 変え られる か が 分から ず 、 外 の 目 も 気 に なり 動か ない 人 も 多い 。 

      　 「 で も 動け ば 主婦 だ って やれる ん です 。 だっ たら 変え たい と 思う 人 に 変える ノウハウ を 伝え て 一緒 に 進め て いこう と 促せ ば いい 」 

      　 昨年 ２ 月 、 その ため の プロジェクト 団体 「 まんま 」 を 発足 さ せ た 。 以来 、 母親 たち が 立ち上がり 、 農業 や 食 を 考え 、 取り組む グループ が 各地 で 生まれ て いる 

      。 

      　 自分 も 動き を 加速 し た 。 

      　 「 今年 、 山 の 田んぼ で 仲間 と 自然 栽培 の 米 づくり を し まし た 。 オーガニック 給食 の 米 が 足り ない と いう の で 、 市民 に も 協力 し て もらい 給食 用 に 作っ た ん です 

      」 

      　 収穫 量 は 給食 ２ 日 分 の ５４０ キロ 。 市 に 納入 し 今月 、 従来 の 有機米 １ 日 分 と 合わせ ３ 日間 の オーガニック ご飯 給食 を 実現 さ せ た 。 

      　 「 みんな で わくわく 一緒 に やる こと が 大事 。 それ で 毎月 １ 回 の 実施 に し 、 毎週 １ 回 に 増やせれ ば … 」 



      【 プロフィール 】 ふる た ・ たか こ 　 滑川市 出身 。 自然 栽培 給食 プロジェクト 団体 「 まんま 」 と 、 その 実動 部隊 「 ごちそう プロジェクト 」 の 代表 。 昨年 秋 に 東京 

      で あっ た 全国 オーガニック 給食 フォーラム で 締めくくり の オーガニック 給食 宣言 を 読み上げ た 。 １０ アール の 畑 で 自然 栽培 も 実践 する 。
