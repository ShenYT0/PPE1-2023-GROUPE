


食品 衛生 監視員 と は 日本 国民 の 食 の 安全 を 守る エキスパート 



--



      「 食品 衛生 監視員 」 は 「 食品 衛生 法 」 など の 法律 に 基づい て 、 私たち が 口 に する 食品 の 安全 と 安心 を 守る スペシャリスト です 。 

      空港 や 港 など の 検疫所 で 働く 「 国家 公務員 」 の 食品 衛生 監視員 と 、 自治体 の 保健所 で 働く 「 地方 公務員 」 の 食品 衛生 監視員 が い ます 。 

--

      もはや 輸入 食品 なし に 国民 の 生活 が 成り立た ない ほど です 。 

      そう し た 輸入 食品 に よる 食 中毒 や 感染症 など の 危険 を 防ぐ ため 、 輸入 食品 の 安全 確保 は 非常 に 重要 な 仕事 です 。 

      その ため 日本 で は 輸入業 者 は 食品 を 輸入 する 時 に は 「 輸入 届出 」 を する こと が 義務 づけ られ て い ます 。 

--



食品 衛生 監視員 は 「 食 の 安全 」 に 興味 が 強い 人 に 向く 



--

      また 食品 の 調査 や 分析 を チーム で 行う の で 協調性 も 大切 です 。 

      「 食 の 安全 」 に 対する 興味 が 強い 人 

      食品 に 関する 法律 や 食品 の 試験 検査 技術 など 、 さまざま な 専門 知識 や 技術 が 必要 と さ れる ため 、 「 食 の 安全 」 に 対する 強い 興味 関心 を 持っ て いる こと が 必須 で 

      す 。 

--

      食品 衛生 監視員 は 国民 や 地域 住民 の ため に 貢献 する と いう 奉仕 の 精神 が 必要 です 。 

      「 日本 の 食 の 安全 は 自分 が 守る 」 と いっ た 強い 責任感 が ある 人 に 向い て い ます 。 

      食品 の 検査 や 分析 が 好き な 人 

--



      食品 衛生 監視員 は 、 国民 の 健康 の ため に 食 の 安全 と 安心 を 守る 仕事 です 。 

      残留 農薬 、 添加物 、 放射性 物質 、 残留 動物 用 医薬品 （ 抗菌性 物質 等 ） など の 悪影響 や 、 疫病 感染 の 危険 から 国民 を 守る ため に 、 今 も 将来的 に も 必要 と さ れる 仕
